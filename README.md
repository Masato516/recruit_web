# 被験者募集サイト

被験者の応募要項を掲載するためのサイトとして作成しました。
被験者を応募するユーザーは学内メールで登録してもらい、募集内容を投稿できるようにする機能になります。
応募する際にはWYSIWYFテキストを用いることができるようにしております。


* Ruby: 2.7.1

* Rails: 5.4.3

* ログイン機能: Devise 4.7.2

* データベース: MySQL 8.0.21

* リッチテキスト: trix-rails 2.2.0

* ファイルアップローダー,バリデーション: Shrine 3.2.2

* 日本語化: rails-18n, devise-i18n

* Bootstrap: 4.3.1

* ページネーション: pagy 3.8

* Webサーバー： Nginx

* ローカル開発環境： Docker

* テスト用ライブラリ： Rspec

* テスト用ドライバー： standalone-chrome

* ファイルストレージ： S3

* サーバー： EC2

* データベース： RDS

* SMTPサーバー： SES

* ロードバランサー： ELB

* ドメインネームシステム： Route53

* SSL/TLS証明書： ACM
